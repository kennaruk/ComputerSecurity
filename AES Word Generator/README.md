AES Word Encryption Generator
====
Problem
----
English
```
Write Java to generator 44 Words by algorithm AES

input are key with 16 characters

Expected Output:
W0 = D1 2B 23 FC
W1 = BB 3C 1A 57
......
W43 = F7 2D FF 2A
```
ไทย
```
ให้เขียนโปรแกรมภาษา C/C++/Java ทำการ generate Word 44 ตัวตาม algorithm ของ AES
โดยโปรแกรมรับ input คือ key (ตัวอักษร 16 ตัว) แล้วใช้ รหัส ASCII ของตัวอักษรเป็นค่าตั้งต้นในการสร้าง W0-W43 แล้วแสดงผลลัพธ์ตามตัวอย่างดังนี้

W0 = D1 2B 23 FC
W1 = BB 3C 1A 57
......
W43 = F7 2D FF 2A

ส่งเฉพาะ Source ในรูปแบบ ass1_ID.c (.cpp or .java) โดย ID คือรหัสนักศึกษา กำหนดส่งภายในวันที่ 20 เมษายน 2561 23:59 (นับเวลาที่เครื่อง server)
```
